  • Chrome: Chrome App Store (Nice and easy, install it from the app store)

  • Firefox: Firefox App Store (Nice and easy, install it from the app store)

Scratch Addons is an browser extension that provides new features and themes for the Scratch website and project editor. It has a mission to collect, archive, and put into use userstyles and userscripts, developed by several members of the Scratch community. Many thanks to @WorldLanguages for putting this awesome extension platform together! For more info visit scratchaddons.com.
